Entrar no mongo shell:
```shell
mongosh
```
Ver as bases de dados:
```shell
show databases
```
Entrar em um banco de dados / criar pela primeira vez:
```shell
use (nome-do-banco)
```
Criar uma coleção:
```shell
db.createCollection(name, options)
```
Excluir uma coleção:
```shell
db.(nomedacoleção).drop()
```
Excluir um banco de dados:
```shell
db.dropDatabase()
```
